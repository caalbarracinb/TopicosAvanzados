3. Algoritmo de descenso del gradiente¶

El algoritmo de descenso del gradiente permite la convergencia hacia un √≥ptimo local por medio de un proceso de ajuste iterativo. La funci√≥n que se quiere optimizar es el Error Cuadr√°tico Medio, o en otras palabras, la distancia entre los valores reales y los predecidos por medio de la regresi√≥n.
$$E = \frac{1}{n}\sum^n_{i=0}(y_i-\hat{y}_i)$$

$$E = \frac{1}{n}\sum^n_{i=0}(y_i-mx_i+c)$$

Como se puede observar, dentro de la funci√≥n existen dos par√°metros para minimizarla, estos son c y m, que ser√≠an el intercepto y la pendiente, respectivamente. Para optimizar la funci√≥n se deven hallar las derivadas parciales respecto a cada uno de ellos.
$$D_m = \frac{-2}{n}\sum^n_{i=0}x_i(y_i-(mx_i+c))$$

$$D_c = \frac{-2}{n}\sum^n_{i=0}-2(y_i-(mx_i+c))$$

Posteriormente, se ajustan los valores iniciales de c y m en funci√≥n al valor del gradiente (las derivadas parciales) y la tasa de aprendizaje. $$m=m-L\times D_m$$ $$c=c-L\times D_c$$ El proceso se repite hasta minimizar la funci√≥n de p√©rdida de acuerdo a un par√°metro establecido, idealmente 0, resultado los ultimos valores de c y m los que √≥ptimos. Si el proceso no converge a un √≥ptimo, el investigador puede establecer un n√∫mero m√°ximo de repeticiones para acabar el proceso.

### Extracci√≥n de las variable para la regresi√≥n 
Y = data['colGPA']
X = data['hsGPA']
# Valores iniciales de los par√°metros
m = 0
c = 0

L = 0.0001  # Tasa de aprendizaje
epochs = 10000  # N√∫mero de iteraciones
criteria = 1e-5 # Criterio para detener las iteraciones
n = float(len(X)) # Number de observaciones

# Performing Gradient Descent 
for i in range(epochs): 
    Y_pred = m*X + c  # Valor predicho actual
    D_m = (-2/n) * sum(X * (Y - Y_pred))  # Gradiente de la pendiente
    D_c = (-2/n) * sum(Y - Y_pred)  # Gradiente de la constante
    m = m - L * D_m  # Update m
    c = c - L * D_c  # Update c
    if max(abs(L * D_m), abs(L * D_c)) < criteria:
        break
# Impresi√≥n de los resultados
print("Los valores que se obtienen son:", m,c, "en",i, "iteraciones")
Y_opt=Y_pred

5. Conclusiones¶

La Regresi√≥n Lineal permite obtener la magnitud de la relaci√≥n lineal que asocia a dos o m√°s variables. Adicionalmente, se present√≥ la implementaci√≥n del algoritmo de descenso del gradiente el cu√°l permite aproximar los resultados que se podr√≠an obtener directamente de la derivaci√≥n de los estimadores por medio de m√≠nimos cuadrados ordinarios (MCO). Adem√°s, hay que se√±alar que los resultados de la aplicaci√≥n del descenso del gradiente converge en cierta medida a los resultados obtenidos por MCO cuando hay suficientes iteraciones, el criterio para detener las iteraciones es lo suficientemente peque√±o, parametros que son totalmente arbitrarios, pues los escoge qui√©n programa el algoritmo.
